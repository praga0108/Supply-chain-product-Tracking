# Supply Chain Product Tracking System

## Tech Stack
- Node.js
- Express.js
- PostgreSQL

## Features
- Create Products
- View Products
- Update Product Status
- Delete Products
- Supply Chain Tracking (Manufactured → Shipped → Delivered)

## API Endpoints

GET /products  
POST /products  
PUT /products/:id/status  
DELETE /products/:id  

## Setup

1. Clone repository
2. Install dependencies
